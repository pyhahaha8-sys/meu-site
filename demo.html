🚀 PyBot License System - Demo

📊 Como Usar o Sistema

O sistema de licenciamento permite controlar o acesso ao PyBot através de keys válidas.

1. Iniciar a API

cd license-system/api
npm start

2. Acessar o Dashboard

http://localhost:3001

3. Gerar uma Key

# Via terminal
node gerarKey.js ouro

# Via web: usar o dashboard
